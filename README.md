# pyglets

#### 介绍
用pyglet开发游戏

#### 软件架构
- python 3.6.6
- pyglet 1.3.2


#### 安装教程
pip install pyglet==1.3.2

#### 文件介绍
1. game1  第一个游戏：井字棋
2. game2  第二个游戏：贪吃蛇


#### game1介绍

- 目录结构


```
- resources  资源文件，包含背景、圈和叉三张图
- t3         游戏源码
  - elements.py   游戏元素，主要是加载三张图
  - main.py       游戏主程序
```
